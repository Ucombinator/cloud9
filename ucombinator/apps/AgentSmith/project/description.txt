AgentSmith is an antivirus program for Android. It can scan Applications, 
Files and Contacts on the system on demand as well as Clipboard contents 
on copy and phone numbers on calls and new Applications on install. It has 
a large database of signatures and will alert the user of malicious activity
via notifications and can request the deletion of detected malicious
applications and contacts.
